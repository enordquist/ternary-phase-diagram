I have a simple plot illustrating the ternary phase diagram in 3d space that you can rotate.

There is a package that makes generating 2d diagrams easy and beautiful! Install python-ternary, see [here](https://github.com/marcharper/python-ternary)
```
conda create -n ternary
conda activate ternary
conda config --add channels conda-forge
conda install python-ternary
```
